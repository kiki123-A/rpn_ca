# RPN Manual

一个关于逆波兰表示法（RPN）的文档手册，使用Nim语言生成。

## 项目描述

这个项目提供了逆波兰表示法（Reverse Polish Notation）的相关文档和手册。RPN是一种数学表达式表示法，其中运算符位于其操作数之后，也被称为后缀表示法。

## 功能特点

- 完整的RPN文档和说明
- 响应式网页设计，支持移动设备
- 深色/浅色主题切换
- 搜索功能
- 文档内容分组浏览

## 技术栈

- **生成工具**: Nim
- **前端**: HTML5, CSS3, JavaScript
- **字体**: Google Fonts (Lato, Source Code Pro)
- **样式**: 自定义CSS
